DeepEnds version 1.0
====================

* To launch within Visual Studio go to View >> Other Windows >> DeepEnds.
* Select either 
** a mix of .NET assemblies and C# projects to load.
** C++ projects to load.
* Create either 
** a DGML file to view within Visual Studio
** or a HTML file containing cyclometric complexity values and the underlying links.
   See http://www.codeproject.com/Articles/1098935/As-Is-Software-Architecture for further details of the concepts.
       http://www.codeproject.com/Articles/1115494/Dive-into-Architecture-with-DeepEnds for further details of usage.
